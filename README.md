
A python program that stores this book information:  
Title, Author  
Year, ISBN  

Executable program with local data base and GUI

Database is sqlite3  
Graphics made using tkinter  

User can:  
1. View all records  
2. Search an entry  
3. Add entry  
4. Update entry  
5. Delete  
6. Close
